객체 (Object)

    - 속성과 기능으로 이루어진 것
      속성 == 변수(필드) == 값
      기능 == 함수(메서드)

JavaScript Object

    - {} (블록) 내에 key : value 형태의 한 쌍으로
      데이터/함수가 저장된 형태
      
    - key : 변수명(필드명), 함수명(메서드명) ==> 이름
    - value : 변수에 저장된 값, 함수 정의    ==> 대입되는 것
    
    - { k:v, k:v, k:v, ... }
    
    ----------------------------------------------------------
    
    [JS 객체 생성 방법]
    
    1. {} : JS 객체 리터럴을 이용한 생성 방법
    
    2. new Object() : 새 객체 생성 (=== {})
    
    3. 생성자 함수 : 생성자 정의 + new 연산자로 객체 생성
    
    * 리터럴(Literal)
    - 고정된 값을 나타내기 위한 표기법
      == 코드에 직접 값을 작성하는 방법
      
      string -> "", ''
      number -> 3, +3, -3, 3.0
      boolean -> true, false
      array -> []
      object -> {}
